== Description ==

Analyze SQL queries directly in WordPress admin:

* Run EXPLAIN and EXPLAIN ANALYZE on SELECT queries
* View table structures, columns, and indexes
* Analyze multiple queries in one session
* Export formatted reports for LLMs (copy or download)
* REST API for programmatic access

Location: **Tools > SQL Analyzer** (requires `manage_options` capability)

== Installation ==

1. Install the plugin.
2. Activate the plugin.
3. Go to **Tools > SQL Analyzer**.

== Frequently Asked Questions ==

= Do I need an API key? =

No. All analysis runs locally on your WordPress installation.

= Is my data private? =

Yes. Queries are never sent externally. You control what you share.
